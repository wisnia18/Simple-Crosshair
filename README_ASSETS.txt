Assets Instructions

Put your custom image and audio files in the 'assets' directory.

Supported images (PNG format with transparent background):
cross1.png - overrides custom crosshair 1
cross2.png - overrides custom crosshair 2
cross3.png - overrides custom crosshair 3
cross4.png - overrides custom crosshair 4

Supported audio (WAV format):
click.wav - played when clicking UI elements
shoot.wav - played on left mouse click
